Formularios (forms)

los formularios son estructuras para capturar datos del cliente (entiendase cliente por usuarios)

inputs -> busquedas, clicks

form -> get (obtener) -> post (enviar)

obtener datos ej: escribir en un buscador y recibir un dato en contreto devuelta

el placeholder solo se ,uestras antes de comenzar a escribir

label busca en el archivo un elemento id para poder tener el artibuto del mismo






en el input color no tiene sentido utilizar el placeholder ya que no tiene proposito darle un dato al usuarios



el input para tomar fecha puede ser de tipo datetime-local, date ó month





sugerencia: no utilizareste tipo de input ("image")



el artibuto range por defecto va a aparecer en la mitad



El input de tipo tel requiere un patrón (atributo pattern) para validar el dato.
Por ejemplo: pattern="[0-9]{4}-[0-9]{4}"
El inicio ([0-9]{4}) indica que se permiten números del 0 al 9, que deben ser 4, luego se debe escribir un guión y finalmente otros 4 números.

la etiqueta (span) Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea. (no abarcar todo el espacio en la pagina)

diferencias entres checkbox y radio es que en checkbox te deja seleccionar varios valores y radio solo te permite escoger una opcion unica
